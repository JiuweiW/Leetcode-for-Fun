```python
class Solution:
    def bitwiseComplement(self, N: int) -> int:
        if N == 0:
            return 1
        res = 0
        count = 0
        while N != 0:
            if N % 2 == 0:
                res += 2**count
            count += 1
            N = N >> 1
        return res
```
